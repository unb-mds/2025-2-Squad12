# Como você pode contribuir

Você pode contribuir identificando possíveis erros na documentação, sugerindo novas funcionalidades ou realizando alterações no código e no scraping de dados.  

Para isso, siga os passos abaixo.

---

# Abra uma nova Issue, ou use uma já existente

Na seção **Issues** do nosso repositório no GitHub, você encontrará os problemas e melhorias mapeados para o É Crypto.

- Você pode **escolher uma issue já aberta** para trabalhar  
  ou  
- **abrir uma nova issue**, descrevendo:
  - qual problema encontrou **ou** qual melhoria está propondo;
  - contexto (prints, links, exemplos);
  - qualquer ideia inicial de solução que já tenha.

É por meio dessa issue que você registra oficialmente o que pretende fazer e, a partir dela, cria sua **branch** de trabalho, mantendo o histórico do projeto organizado.

---

# Implementando Contribuição

Branch criada e issue atribuída, basta agir.

Você pode usar a IDE de sua preferência para:

- implementar novas funcionalidades;
- corrigir bugs;
- melhorar spiders e pipelines de coleta de dados;
- ajustar a interface ou usabilidade;
- atualizar a documentação (incluindo esta GitPage, quando necessário).

Sempre que possível, faça mudanças claras e coesas, facilitando a revisão depois.

---

# Faça um Pull Request para completar

Depois de concluir as modificações:

1. Envie sua branch para o repositório remoto.
2. Abra um **Pull Request (PR)** direcionado para a branch principal (`main`).

No PR, explique brevemente:

- qual issue está sendo resolvida ou qual melhoria está sendo entregue;
- o que foi alterado;
- se houver, como testar o que você fez.

A partir daí, sua contribuição será analisada, discutida (se necessário) e, se aprovada, será integrada ao projeto.
